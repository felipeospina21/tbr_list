TanStack useSuspenseQuery inside a Next client component can execute during server rendering. Browser-relative API route fetches like /api/reading-list fail on the server with Failed to parse URL and may also miss browser cookies. Gate browser-only suspense queries until client mount, or move the data fetch to a server-safe path.
